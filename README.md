# Tex_projects

- LaTeX color code: color code reference: http://latexcolor.com/
- Tikz manual: https://ctan.math.washington.edu/tex-archive/graphics/pgf/base/doc/pgfmanual.pdf
- Tikz examples: https://texample.net/tikz/examples/all
